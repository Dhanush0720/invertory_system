📁 LOGO FOLDER
==============
Place your college logo image file here.

Supported formats: .png, .jpg, .jpeg, .svg, .webp

Recommended:
  - File name : logo.png  (or logo.svg for best quality)
  - Size      : 200×200 px minimum, square or wide format works best
  - Background: Transparent PNG preferred

Once you add the logo file, it will automatically appear in:
  ✅ Sidebar (top left)
  ✅ Login page (center)

No code changes needed — just drop the file here and name it "logo.png"
(or update the name in src/config/logo.js if you use a different name).
